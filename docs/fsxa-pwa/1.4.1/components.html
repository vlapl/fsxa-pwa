Base Components

Introduction

The FSXA-Pattern-Library provides multiple base components, which come with a lot of attributes and methods to help you build your PWA. This chapter will introduce you to these different components and will describe when and how to use them.

Components

FSXABaseComponent

Basic Component for all further components, provides general attributes and methods.

FSXABaseAppLayout

Basic Component to implement the general layout of the app.

FSXABaseLayout

Basic Component to implement a page layout.

FSXABaseSection

Basic Component to implement a section of a page.

FSXABaseRichTextElement

Basic Component to implement a rich text element.
